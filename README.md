# Paint Android Studio :iphone: :art:

A simple drawing application made with Android Studio.

## Features
- Free drawing
- Line drawing
- Rectangle drawing
- Choice of colour

## Tested with
- Android Studio 2022.2.1
- Virtual Pixel 6 API 33
- Gradle 8.0
